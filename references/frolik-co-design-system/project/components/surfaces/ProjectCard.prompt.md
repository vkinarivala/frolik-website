One-line: Portfolio project card with an image area and meta, lifting to a coral hard-shadow on hover - the core unit of the portfolio grid.

```jsx
<ProjectCard title="The Playhouse Loft" category="Residential" location="Ahmedabad" mediaColor="var(--mint-200)" />
```

Pass an `<img>` as `media` for real photography; otherwise it shows a solid `mediaColor` block.
