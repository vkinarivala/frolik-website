## Technical SEO checklist

- [ ] `sitemap.xml` at root (generated — see file); reference it in `robots.txt` via `Sitemap: https://frolik.co.in/sitemap.xml`.
- [ ] `robots.txt` allowing all, disallowing any `/thank-you` / form-success pages.
- [ ] **LocalBusiness** schema on Home & Contact (name, address, Ahmedabad service area, phone, geo, opening hours, sameAs → Instagram/Google Business Profile).
- [ ] **FAQPage** schema on `/kids-spaces` and each kids landing page.
- [ ] **BreadcrumbList** schema on Work, Kids Spaces, Services subpages.
- [ ] Canonical tags on every page; category filter views canonical to `/work`.
- [ ] Descriptive image alt text, e.g. *"Pastel kids room interior designed by Frolik + Co in Ahmedabad."*
- [ ] Click-to-call `tel:` and `wa.me` WhatsApp links in header/footer.
- [ ] Open Graph + Twitter card meta on all pages (project hero as OG image).
- [ ] One H1 per page; H2/H3 hierarchy follows the content plan.
- [ ] "Ahmedabad" present in title tags of location-intent pages.
- [ ] Internal links: Home → Kids Spaces + Work; Kids Spaces ↔ related projects; projects → related projects.

> Update `lastmod` in `sitemap.xml` whenever a page's content changes; regenerate on each project/journal publish.
